\section{OS Image}
\begin{enumerate}[1.]
\item Download the Ambian Bionic Desktop OS image (kernel 4.4) from 
\begin{lstlisting}
https://www.armbian.com/tinkerboard/
\end{lstlisting}
and extract the .img file.
\item Burn the image to the sdcard 
\begin{lstlisting}
sudo dd if=armbianos.img /dev/sdX bs=4096 status=progress
\end{lstlisting}
where the sdcard location /dev/sdX can be obtained thorugh 
\begin{lstlisting}
sudo fdisk -l
\end{lstlisting}
%
\item Inserd the sdcard with the armbian bionic image in the tinkerboard
and power on. 
\item Enter login as \textbf{root} and password \textbf{1234}.  Choose a \textbf{username}
and \textbf{password} for yourself. You should boot into the desktop.
\item Set timezone
\begin{lstlisting}
sudo dpkg-reconfigure tzdata
\end{lstlisting}
\end{enumerate}

\section{Wifi on Armbian Bionic desktop mainline kernel 4.14.y}
Wifi doesn't work properly with the 4.14 Ubuntu desktop due to \textbf{Network Manager}. To resolve this do the following.
\begin{enumerate}[1.]
\item Install wpagui
\begin{lstlisting}
sudo apt install wpagui
\end{lstlisting}
\item Remove network manager
\begin{lstlisting}
sudo apt remove network-manager
\end{lstlisting}
\item Replace interfaces
\begin{lstlisting}
wget ifaces.txt 
cp ifaces /etc/network/interfaces
\end{lstlisting}
\item Open wpa\_supplicant configuation file
\begin{lstlisting}
/etc/wpa_supplicant/wpa_supplicant.conf
\end{lstlisting}
and type the following in the beginning
\begin{lstlisting}
ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
update_config=1
\end{lstlisting}
\item Add user to group netdev
\begin{lstlisting}
sudo usermod -a -G netdev $USER
\end{lstlisting}
\item Check wpa\_supplicant status
\begin{lstlisting}
sudo systemctl status wpa_supplicant.service
\end{lstlisting}
If active, do nothing.  Else start it 
\begin{lstlisting}
sudo systemctl status wpa_supplicant.service
\end{lstlisting}
\item For giving wpa\_supplicant access to wifi network interface,
\begin{lstlisting}
sudo wpa_supplicant  -i wlan0 -c/etc/wpa_supplicant/wpa_supplicant.conf -D wext 
\end{lstlisting}
\item Open wpa-gui from application menu.  Scan for networks. Select your preferred network
and enter the password. Connect.  In case connection errors are displayed, reboot.

\end{enumerate}
